\subsection{Тензор энергии-импульса}

    Имеет вид:
    %
    \begin{equation}
        T^\mu_\nu = \pdv{\mathcal{L}}{h_{ij,\mu}} h_{ij,\nu} - \mathcal{L} \delta^\mu_\nu
    \end{equation}

    Компонента $T^0_0$ определяет плотность энергии. Вектор потока складывается из компонент $T^i_0$.
